# 🚙 Land Cruiser

A premium luxury-inspired **Land Cruiser Web Experience** built using **Node.js, Express.js, MongoDB, EJS, Tailwind CSS, and JavaScript**.

Designed with a modern UI, responsive layout, and smooth user experience to showcase Land Cruiser models in a premium and elegant way.

---

## ✨ Features

- 🌙 Luxury Dark Theme UI
- 📱 Fully Responsive Design
- ✨ Glassmorphism Navbar
- 🎨 Modern Tailwind CSS Design
- 🖼 Beautiful Hero Section
- 🚗 Explore Land Cruiser Models
- 📄 Dynamic EJS Templates
- 🗄 MongoDB Database Integration
- 🔐 User Authentication (Login & Register)
- ⚡ Fast Express Server
- 💻 Clean Project Structure

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/Land-Cruiser.git
```

Move into the project

```bash
cd Land-Cruiser
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
PORT=3000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

Start the server

```bash
npm run dev
```

or

```bash
npm start
```
